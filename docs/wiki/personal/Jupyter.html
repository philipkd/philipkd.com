Environment variables

IPython

from IPython.display import display, Markdown, Audio, Image

Running other notebooks

%run runs another notebook as if the code was copied and pasted

Running on Server

remote: jupyter lab --NotebookApp.token=""

local: ssh -N -f -L localhost:8888:localhost:8888 ubuntu@IP_ADDRESS

With Colab front-end

jupyter lab --NotebookApp.allow_origin='https://colab.research.google.com' --NotebookApp.token=""

--NotebookApp.port_retries=0 and --no-browser might also be necessary

Timing operations

%%time shows wall time (refers to IRL clock on wall) for entire cell.
%time shows for just one line.

Config

jupyter notebook --generate-config then

c.LabApp.token = '' c.LabApp.password = HASH c.LabApp.open_browser = False c.LabApp.notebook_dir = '...'

To get the hash, run in Python: from notebook.auth import passwd; passwd()

To edit the indent size: Settings > Advanced Settings > Notebook
