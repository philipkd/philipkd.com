Mounting Google Drive

When you press the Drive icon, either of two things happen:

  1. A cell is created:

from google.colab import drive drive.mount('/content/drive')

  1. A connection to Drive is made

No. 2 is better because when you run the notebook, you don't have to Oauth each time. The method of pairing is tied to the notebook, and it's uncertain what determines which notebook does which behavior. However, the good type of pairing always works for new, blank notebooks, but not if you import a notebook.

Workaround:

  1. create a blank notebook
  2. connect to Drive
  3. import a notebook in a new window
  4. select all cells
  5. paste all cells in the original notebook (tooltip will pop up saying to hit Cmd+V)
