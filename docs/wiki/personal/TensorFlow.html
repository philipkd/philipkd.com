eager execution

TensorFlow functions execute operations immediately (as opposed to adding to a graph to be executed later in a tf.compat.v1.Session) and return concrete values (as opposed to symbolic references to a node in a computational graph).

TFRecords

protocol buffers are an XML-like data format created by Google, but designed to generate native code with the data. "protocol message" refers to the data in an abstract sense, while the "buffer" refers to the serialized copy of the message. source
